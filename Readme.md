
CLI tool for Zox.js

[Official Zox.js site](http://zoxjs.com)

### Install

```bash
npm i -g zox-cli
```

### Usage

```bash
zox init <template-name> <project-name>
```

### Example:

```bash
zox init static-site-handlebars my-site
```

### Custom templates

```bash
vue init <username>/<repo> <my-project>
```
